DoceboRuby - Ruby wrapper for Decobo LMS API
============================

### Original Author
Cẩm Huỳnh, https://github.com/huynhquancam/docebo_ruby

### Installation

```sh
gem install docebo
```

or add it to Gemfile

```
gem 'docebo', '~> 0.0.1'
```

### Usage


```ruby
api = Docebo::API.new(
  key: 'key', 
  secret: 'secret',
  url: 'your_saas_platform/api'
)

results = api.make_request(
  'user', 
  'authenticate',
  username: 'hello', 
  password: 'hell'
)
```
